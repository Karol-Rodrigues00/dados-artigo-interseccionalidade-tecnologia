# Dados suplementares do artigo

Este repositório contém os materiais utilizados no artigo:

**"Para Além do Gênero: Mapeamento Sistemático da Literatura sobre Interseccionalidade, Exclusão e Permanência Feminina na Tecnologia da Informação"**

## Conteúdo
- Planilhas 
- fichas de extração dos estudos
- Gráficos e tabelas utilizadas na análise
- Fluxograma

## Objetivo
Garantir transparência, reprodutibilidade e acesso aberto aos dados da pesquisa.
